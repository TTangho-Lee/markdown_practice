# Markdown.kr에 오신 것을 환영합니다!

**Markdown.kr**은 설치 없이 브라우저에서 바로 쓰는 **무료 온라인 마크다운 에디터**입니다.
왼쪽에 마크다운 문법으로 입력하면 오른쪽 미리보기가 실시간으로 업데이트됩니다.

> **모바일 사용자** — 툴바 왼쪽의 파란색 **`미리보기`** 버튼을 눌러 \
에디터와 미리보기 화면을 전환하세요.

---

## 파일 관리  _(상단 파일 메뉴)_

| 기능 | 단축키 | 설명 |
|---|---|---|
| 새 파일             | `Ctrl+N` | 현재 내용을 비우고 새로 시작 |
| 파일 열기 | `Ctrl+O` | 컴퓨터의 .md / .txt 파일 불러오기 |
| MD 저장 | `Ctrl+S` | 마크다운 파일(.md)로 다운로드 |
| HTML 내보내기 | — | 렌더링된 HTML 파일로 저장 |
| PDF 내보내기 | — | PDF 파일로 저장 |

---
`p`

## Unmarkdown  _(상단 변환 메뉴)_

마크다운 기호(`#`, `*`, `` ` ``, `>` 등)를 **모두 제거한 순수 텍스트 .txt** 파일로 내보냅니다.
이메일 본문, 워드 문서, 노션, 슬랙 등에 바로 붙여넣을 때 유용합니다.

---

## 기타 기능

- **구문 강조** — 에디터에서 마크다운 문법이 색상으로 표시되어 가독성이 높아집니다
- **자동 저장** — 브라우저를 닫아도 마지막 작성 내용이 유지됩니다
    - **다크 모드** — 상단 달 아이콘으로 다크/라이트 테마를 전환합니다
        - **마크다운 가이드** — 상단 메뉴의 `가이드`에서 전체 문법을 확인할 수 있습니다
         - *피곤하다*
         - **자고싶다**
